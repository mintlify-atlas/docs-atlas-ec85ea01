## Code Contributions

If you're a developer, you can help by fixing bugs, adding new features, or improving the code quality.

### Getting Started

1. Fork the [FixTweetBot repository](https://github.com/Kyrela/FixTweetBot)
2. Create a new branch for your feature or bugfix
3. Make your changes following the project's coding standards
4. Test your changes thoroughly
5. Submit a pull request

### Contributor License Agreement

Before your pull request can be merged, you must sign the Contributor License Agreement (CLA). This ensures that your contributions can be distributed under the project's license.

<Warning>
  To accept the CLA, comment on your pull request with:
  
  **"I have read the CLA Document and I hereby sign the CLA"**
</Warning>

Read the full [CLA document](https://github.com/Kyrela/FixTweetBot/blob/main/CLA.md) to understand the terms.

### Platform Compliance

When contributing code, ensure that:
- Your code does not violate any third-party platform's terms of service
- You follow best practices for API usage and rate limiting
- You respect user privacy and data protection

## Translations

Help make FixTweetBot accessible to users worldwide by contributing translations.

### Currently Supported Languages

The bot is currently available in:
- English (en)
- French (fr)
- Polish (pl)
- Russian (ru)
- Spanish - Spain (es-ES)
- Vietnamese (vi)
- Chinese - Simplified (zh-CN)
- Chinese - Traditional (zh-TW)
- Portuguese - Brazil (pt-BR)
- Japanese (ja)

### How to Translate

<Steps>
  <Step title="Join Crowdin">
    Visit the [FixTweetBot Crowdin project](https://crowdin.com/project/fixtweetbot)
  </Step>
  <Step title="Select a Language">
    Choose your language or request a new one if it's not listed
  </Step>
  <Step title="Start Translating">
    Translate strings and submit your suggestions
  </Step>
  <Step title="Review">
    Help review other contributors' translations
  </Step>
</Steps>

Your translations will be included in the next bot release once they're approved.
