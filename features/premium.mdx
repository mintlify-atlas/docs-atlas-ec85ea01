## Bot & Webhook Response

The standout premium feature is the ability to respond to messages from bots and webhooks.

### How It Works

<Steps>
  <Step title="Enable in Settings">
    After subscribing to Premium, enable the "Respond to bots/webhooks" option in `/settings`.
  </Step>
  
  <Step title="Automatic Detection">
    FixTweetBot will now process links from:
    - Other Discord bots
    - Webhook messages
    - Automated posting systems
  </Step>
  
  <Step title="Same Great Experience">
    Links from bots are fixed the same way as user messages, with all your configured settings applied.
  </Step>
</Steps>

### Use Cases

<AccordionGroup>
  <Accordion title="RSS Feed Bots" icon="rss">
    Many servers use RSS bots to automatically post content from social media accounts. Premium allows FixTweetBot to fix these links automatically.
    
    **Example:** Twitter RSS bot posts new tweets → FixTweetBot fixes them instantly
  </Accordion>
  
  <Accordion title="News & Update Bots" icon="newspaper">
    News bots often share links to social media posts. With Premium, these are automatically enhanced.
    
    **Example:** Gaming news bot shares TikTok → FixTweetBot provides better embed
  </Accordion>
  
  <Accordion title="Webhook Integrations" icon="webhook">
    If you use webhooks to integrate external services, Premium ensures all links are fixed.
    
    **Example:** GitHub webhook shares Instagram link → FixTweetBot fixes it
  </Accordion>
  
  <Accordion title="Multi-Bot Servers" icon="bots">
    Servers with multiple bots benefit from seamless link fixing across all bot messages.
    
    **Example:** Any bot posts social media link → FixTweetBot handles it
  </Accordion>
</AccordionGroup>

### Configuration

The bot/webhook response feature is configured in the Guild model:

```python
__casts__ = {
    'webhooks': bool,
    # ... other settings
}
```

This setting determines whether the bot processes messages from non-user sources.

## Premium FAQ

<AccordionGroup>
  <Accordion title="Is Premium required to use the bot?">
    No, FixTweetBot is free to use with all core features available. Premium only adds the ability to respond to bots/webhooks and provides priority support.
  </Accordion>
  
  <Accordion title="Can I try Premium before buying?">
    Unfortunately, there's no free trial available. However, the Premium feature set is clearly defined, and you can see exactly what you're getting before subscribing.
  </Accordion>
  
  <Accordion title="What happens if I cancel Premium?">
    If you cancel your Premium subscription:
    - The bot will stop responding to bots and webhooks
    - All other features continue to work normally
    - Your server settings are preserved
    - You can resubscribe at any time
  </Accordion>
  
  <Accordion title="Can I gift Premium to another server?">
    Currently, Premium subscriptions are tied to the purchasing server. Contact support if you need special arrangements.
  </Accordion>
  
  <Accordion title="How do I manage my subscription?">
    Premium subscriptions are managed through Discord's subscription system. Visit your server settings → Subscriptions to manage your FixTweetBot Premium.
  </Accordion>
  
  <Accordion title="Do I get a refund if I'm not satisfied?">
    Refunds are handled by Discord's subscription system. Check Discord's refund policy for details.
  </Accordion>
</AccordionGroup>
