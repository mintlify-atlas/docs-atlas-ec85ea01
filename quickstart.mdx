# Quickstart guide

Get FixTweetBot up and running in your Discord server in just a few steps.

<Steps>
  <Step title="Invite the bot to your server">
    Click the invite link to add FixTweetBot to your Discord server:
    
    [Invite FixTweetBot](https://discord.com/oauth2/authorize?client_id=1164651057243238400)
    
    <Note>
    You need **Manage Server** permission to add bots to a Discord server.
    </Note>
    
    Select the server from the dropdown and authorize the bot with the requested permissions:
    
    - View Channels
    - Send Messages
    - Send Messages in Threads
    - Embed Links
    - Read Message History
    - Manage Messages (to suppress/delete original embeds)
    
    <Warning>
    The bot requires **Manage Messages** permission to remove or suppress embeds from your original messages. Without this, the bot will still post fixed links but can't clean up the original.
    </Warning>
  </Step>

  <Step title="Send a social media link">
    Post a message containing a link from any supported platform. For example:
    
    ```
    Check out this tweet: https://twitter.com/user/status/1234567890
    ```
    
    Or try Instagram, TikTok, Reddit, or any of the 30+ supported platforms:
    
    ```
    https://www.instagram.com/p/ABC123/
    https://www.tiktok.com/@user/video/1234567890
    https://www.reddit.com/r/example/comments/abc123/
    ```
    
    <Info>
    The bot automatically detects links in regular messages. You don't need to use any special commands or syntax.
    </Info>
  </Step>

  <Step title="Bot posts the fixed link">
    FixTweetBot will:
    
    1. Detect the social media link in your message
    2. Reply with a fixed version that has better embeds
    3. Suppress or delete the original embed (if configured and bot has permissions)
    
    The fixed message includes:
    - A link to the fixed embed (better video playback, galleries, etc.)
    - A link to the original post
    - A link to the author's profile (when available)
    
    Example output:
    ```
    [Tweet](https://twitter.com/user/status/1234567890) • [@user](https://twitter.com/user) • [FxTwitter](https://fxtwitter.com/user/status/1234567890)
    ```
  </Step>

  <Step title="Customize settings (optional)">
    Use the `/settings` command to customize how the bot behaves:
    
    ```
    /settings
    ```
    
    This opens an interactive menu where you can:
    
    - Enable/disable specific platforms
    - Choose embed view styles (normal, direct media, gallery, text-only)
    - Set whether to keep, suppress, or delete original messages
    - Configure silent replies
    - Set up filters by channel, role, or member
    - Add keyword filters
    - Enable post translation
    - Add custom websites
    
    <Tip>
    The `/settings` command requires **Manage Messages** permission in your server.
    </Tip>
  </Step>
</Steps>

## Ignoring specific links

If you want the bot to skip a particular link, add `fxignore` anywhere in your message:

```
Check this out https://twitter.com/user/status/123 fxignore
```

The bot will ignore all links in that message and leave the normal embed.

## Common issues

<AccordionGroup>
  <Accordion title="Bot doesn't respond to my links">
    Check that:
    1. The bot has permission to view the channel (View Channels)
    2. The bot has permission to send messages (Send Messages, Embed Links)
    3. The platform is enabled in `/settings` > Websites
    4. The channel isn't filtered out in `/settings` > Filters
    5. Your message doesn't contain `fxignore`
  </Accordion>
  
  <Accordion title="Original embed doesn't get removed">
    The bot needs **Manage Messages** permission to suppress or delete the original embed. Check the bot's role permissions in Server Settings > Roles.
  </Accordion>
  
  <Accordion title="Fixed link has no embed">
    This can happen if:
    - The fixer service is temporarily down
    - The original post was deleted or made private
    - The content exceeds Discord's embed size limits
    
    The bot will automatically delete failed fixed messages after a few seconds.
  </Accordion>
  
  <Accordion title="Want to respond to other bots or webhooks?">
    This is a premium feature. Check out the [premium subscription](https://discord.com/application-directory/1164651057243238400/store/1260699263755751456) or use the `/about` command to learn more.
  </Accordion>
</AccordionGroup>
